# Data Collection
## We collect the following types of data from our users:
Data_Collected: Personal Information, Cookies, IP Address, Usage Data, Transaction Data

## Personal Information: Includes names, email addresses, phone numbers, and mailing addresses provided by the user during account creation, purchase, or contact forms.
## Cookies: We collect cookies to track session information, preferences, and for analytics purposes.
## IP Address: Your IP address is collected when you visit our site for fraud prevention, geolocation, and analytics.
## Usage Data: We track user interactions on our site, including pages visited, time spent, and links clicked for performance optimization.
## Transaction Data: Payment information is collected for processing transactions (Note: Financial data is encrypted and stored by our payment processor).

# User Rights
## In compliance with global privacy laws, users have the following rights regarding their data:
User_Rights: Access, Correction, Deletion, Portability, Restriction, Objection

## Access: Users can request access to the data we store about them.
## Correction: Users can request that incorrect or incomplete data be updated.
## Deletion: Users have the right to request the deletion of their data (with exceptions as required by law).
## Portability: Users can request a copy of their data in a machine-readable format.
## Restriction: Users can request that we limit the use of their personal data under specific circumstances.
## Objection: Users have the right to object to the processing of their data for marketing purposes or under certain legal grounds.

# Cookies
## The following types of cookies are used on our site:
Cookies_Used: Analytical, Functional, Advertising, Session

## Analytical Cookies
## Purpose: These cookies are used to track user interactions, website traffic, and performance metrics. They help us understand how visitors engage with our site so we can improve usability and content.

1. **Google Analytics (https://analytics.google.com)**
   - Cookie Name: `_ga`, `_gid`, `_gat`
   - Purpose: Tracks user behavior, such as page views, time on site, and interactions. Data is anonymized where possible, and retention is set for 26 months.
   - Data Collected: Anonymized IP address, browsing activity, device type.
   - Opt-out: Users can opt out via the Google Analytics Opt-out Browser Add-on (https://tools.google.com/dlpage/gaoptout).

2. **Hotjar (https://www.hotjar.com)**
   - Cookie Name: `_hjIncludedInSample`, `_hjSessionUser`, `_hjID`
   - Purpose: Collects data about how users interact with individual pages on our site through heatmaps, session recordings, and feedback polls.
   - Data Collected: Session behavior, scroll depth, click positions, anonymized IP address.
   - Opt-out: Users can opt-out via Hotjar’s Do Not Track feature (https://www.hotjar.com/legal/compliance/opt-out).

3. **Microsoft Clarity (https://clarity.microsoft.com)**
   - Cookie Name: `_clck`, `_clsk`
   - Purpose: Provides insights into user interaction, such as clicks, page scrolls, and session duration, to improve user experience.
   - Data Collected: Anonymized data on user interactions with the website, heatmaps, session recordings.
   - Opt-out: Users can opt-out using browser settings or Clarity’s privacy settings.

## Functional Cookies
## Purpose: Functional cookies store information to remember user preferences, such as language, region, or theme settings. These cookies are essential to providing a customized experience but do not track personal data.

1. **WPML (WordPress Multilingual Plugin) (https://wpml.org)**
   - Cookie Name: `_icl_current_language`, `_icl_visitor_lang_js`
   - Purpose: Stores the user’s preferred language settings to ensure the correct language is displayed on subsequent visits.
   - Data Collected: Language preference.
   - Retention: 24 hours.

2. **OneTrust (https://www.onetrust.com)**
   - Cookie Name: `OptanonConsent`, `OptanonAlertBoxClosed`
   - Purpose: Stores user consent settings for cookies and other tracking mechanisms based on the consent given during their first visit.
   - Data Collected: User cookie consent preferences.
   - Retention: 12 months.

3. **Cloudflare (https://www.cloudflare.com)**
   - Cookie Name: `__cfduid`
   - Purpose: Used to identify trusted web traffic and to protect against malicious visitors.
   - Data Collected: Anonymized user session data for security purposes.
   - Retention: 30 days.

## Advertising Cookies
## Purpose: These cookies are used to deliver personalized ads and track the performance of advertising campaigns. They help us target ads to users based on their browsing behavior.

1. **Facebook Pixel (https://www.facebook.com/business/help/742478679120153)**

   - Data Collected: User behavior, browsing activity, Google account information if logged in.
   - Opt-out: Users can opt-out via Google’s Ads Settings (https://adssettings.google.com).

3. **Criteo (https://www.criteo.com)**
   - Cookie Name: `criteo_id`, `cto_lwid`
   - Purpose: Provides personalized product recommendations and retargeting ads based on user’s previous interactions on our website and across Criteo’s advertising network.

4. **LinkedIn Ads (https://www.linkedin.com)**
   - Cookie Name: `li_sugr`, `UserMatchHistory`
   - Purpose: Tracks user behavior on the website to deliver personalized ads and measure ad performance for LinkedIn campaigns.

   - Opt-out: Users can manage ad settings via LinkedIn’s Privacy Settings (https://www.linkedin.com/psettings/advertising).

## Session Cookies
## Purpose: These temporary cookies are necessary for the website to function properly. They help manage user sessions, logins, and form submissions. These cookies do not collect personal data and are automatically deleted when the session ends.

1. **PHP Session Cookie (https://www.php.net)**
   - Cookie Name: `PHPSESSID`
   - Purpose: Manages user session across different pages of the website to maintain logged-in status and user-specific interactions.
   - Data Collected: Temporary session ID, no personal data is stored.
   - Retention: Deleted when the session ends (browser is closed).

2. **WooCommerce (https://woocommerce.com)**
   - Cookie Name: `woocommerce_cart_hash`, `woocommerce_items_in_cart`
   - Purpose: Maintains the contents of the shopping cart for users while browsing the website. No personally identifiable information is stored.
   - Data Collected: Session ID, cart contents.
   - Retention: Deleted when the session ends (browser is closed).

3. **WordPress (https://wordpress.org)**
   - Cookie Name: `wordpress_logged_in_`, `wp-settings-1`
   - Purpose: Manages the logged-in status for users who have an account on the site, ensuring they remain authenticated across different pages.
   - Data Collected: Session ID for logged-in users.
   - Retention: Deleted when the session ends (browser is closed).
